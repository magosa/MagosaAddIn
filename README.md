# MagosaAddIn - PowerPoint図形操作補助アドイン

PowerPointでの図形操作を効率化するVSTOアドインです。図形の整列・分割・配置・選択補助・ハンドル調整・レイヤー管理・ナンバリング・置き換え・サイズ調整・配列複製・テーマカラー生成・テキスト一括編集・スタイルライブラリ機能を提供します。

## 🚀 機能概要

### 📐 図形整列機能
- **基準整列**: 左端・右端・上端・下端・水平中央・垂直中央揃え
- **隣接整列**: 基準図形に隣接した配置
- **拡張整列**: 等間隔配置・マージン配置・グリッド配置・円形配置

### ✂️ 図形分割機能
- **単一図形分割**: 1つの四角形を指定グリッドに分割
- **複数図形グリッド分割**: 複数図形の範囲内でグリッド分割
- **スタイル保持**: 元図形の書式を新しい図形に継承

### 🎯 選択補助機能
- **同一書式選択**: 基準図形と同じ書式を持つ図形を自動選択
- **選択条件**: 塗りつぶし色・枠線スタイル・シェイプ種類から選択可能
- **リアルタイムプレビュー**: 条件変更時に一致する図形数を表示
- **選択オブジェクトスタック**: 複数の選択状態を保存・復元
  - 最大20個のスタックを管理
  - スタックごとに図形の選択状態を記憶
  - ワンクリックで過去の選択状態を復元
  - 削除された図形は自動的に無効として表示

### 🔧 図形ハンドル調整機能
- **調整ハンドル設定**: 角丸・矢印・星形などの図形パラメータを一括調整
- **角度ハンドル設定**: 円弧・扇形・ブロック円弧などの角度を度数単位で設定
- **単位変換対応**: mm単位、度数単位での直感的な値入力
- **調整リセット**: すべての調整ハンドルをデフォルト値に戻す

### 📚 レイヤー（重なり順）調整機能
- **選択順配置**: 選択した順番に前面・背面へ配置
- **位置基準配置**: 左から右、上から下の順に前面へ配置
- **重なり制御**: 複雑な図形の重なり順を簡単に整理

### 🔢 自動ナンバリング機能
- **多様なフォーマット**: 算用数字、丸数字、アルファベット、ローマ数字
- **柔軟な設定**: 開始番号・増分値・フォントサイズを指定可能
- **テキスト継承**: 既存テキストの先頭に番号を追加
- **単一図形対応**: 1個の図形からナンバリング可能

### 🔄 図形置き換え機能
- **一括置き換え**: 複数図形をテンプレート図形で一括置換
- **サイズ選択**: 元のサイズ維持 or テンプレートサイズ統一
- **属性継承**: スタイル（塗り・線・影）とテキストの継承オプション
- **中心位置保持**: 置き換え後も図形の中心位置を維持

### 📏 図形サイズ調整機能
- **基準サイズ適用**: 最初の図形を基準に、他の図形を同じサイズに調整
- **比率保持リサイズ**: 幅または高さを統一し、アスペクト比を保持
- **最大/最小サイズ統一**: 選択図形の中で最大または最小のサイズに統一
- **パーセント拡大縮小**: 全図形を指定%（10～500%）で拡大縮小
- **固定サイズ設定**: mm/cm/pt単位でサイズを直接指定
- **中心位置保持**: リサイズ後も図形の中心位置を維持

### 🔁 配列複製機能
- **線形配列**: 指定方向・個数・間隔で図形を複製配列
- **円形配列**: 等分配置または角度指定で円形に配列
- **グリッド配列**: 行×列で格子状に配列、角度回転も可能
- **パス配列**: カスタムパス（線）に沿って配列
- **回転コピー**: 指定角度ずつ回転させながら複製

### 🎨 テーマカラー生成機能
- **17種類の配色パターン**: 色相ベース、トーンベース、コントラスト配色
- **色空間変換**: RGB⇔HSL⇔HSV変換、16進数カラーコード対応
- **明度バリエーション**: 基準色から明暗方向に段階的展開（1～10段階）
- **カラーパレット配置**: スライド枠外に自動配置
- **図形への色適用**: 選択図形に順番またはランダムで色を適用

### ✏️ テキスト一括編集機能 *(v1.0.9 新機能)*
- **テキスト一覧編集**: 選択図形のテキストをグリッドで一覧表示・直接編集
- **全図形一括設定**: 全図形に同じテキストをワンクリックで設定
- **テキスト配布**: 改行区切りのテキストを各図形に1行ずつ順番に配布
- **テキスト一括削除**: 選択図形のテキストをすべて空にする
- **検索・置換**: 図形内テキストの検索（大文字小文字区別オプション付き）
- **フォント書式一括変更**: フォント名・サイズ・太字・斜体・下線・色をチェックした項目のみ一括変更
- **テキストレイアウト調整**: 行間・左右上下余白をチェックした項目のみ一括変更
- **フォント一覧**: システムインストール済み全フォントをアルファベット順で表示（オートコンプリート対応）

### 📸 図形スタイルライブラリ機能 *(v1.0.9 新機能)*
- **スタイル保存**: 図形の塗りつぶし・グラデーション・枠線・影・フォントを名前付きで保存
- **ワンクリック適用**: 保存したスタイルを選択図形に一括適用
- **プレビュー表示**: 選択したスタイルをGDI+でリアルタイム描画してプレビュー
- **お気に入り管理**: ★マークでお気に入り登録・フィルター表示
- **名前検索**: スタイル名でインクリメンタル検索
- **インポート/エクスポート**: JSONファイルで保存・共有（チーム間共有対応）
- **永続化**: `%APPDATA%\MagosaAddIn\StyleLibrary.json` に自動保存（最大100件）

## 📋 機能詳細

### ✏️ テキスト一括編集機能（v1.0.9）

#### テキスト編集タブ
| 機能 | 説明 |
|------|------|
| グリッドの内容を適用 | グリッドで直接編集したテキストを各図形に書き込む |
| テキストを一括削除 | 選択図形のテキストをすべて空にする |
| 全図形に同じテキストを設定 | 1行入力 → 全図形に同じ内容を設定 |
| テキストを配布 | 複数行テキストを改行で分割し各図形に1行ずつ配布 |

#### 検索・置換タブ
| 機能 | 説明 |
|------|------|
| すべて置換 | 図形内テキストを検索・置換 |
| 検索のみ | 一致する図形数を表示（置換なし） |
| 大文字/小文字区別 | 検索時の大文字小文字区別オプション |

#### フォント書式タブ
| 設定項目 | 説明 |
|---------|------|
| フォント名 | システム全フォントからオートコンプリートで選択 |
| フォントサイズ | ポイント単位で指定 |
| 太字・斜体・下線 | 各スタイルの適用ON/OFF |
| フォント色 | カラーダイアログで色を選択 |

※ チェックした項目のみ変更（未チェック項目は維持）

#### レイアウトタブ
| 設定項目 | 説明 |
|---------|------|
| 行間 | 倍数で指定（0.5～10倍） |
| 左余白・右余白 | pt単位で指定 |
| 上余白・下余白 | pt単位で指定 |

※ チェックした項目のみ変更（未チェック項目は維持）

### 📸 図形スタイルライブラリ機能（v1.0.9）

#### スタイル情報
保存されるスタイル情報：
| 種類 | 詳細 |
|------|------|
| 塗りつぶし | 単色（色・透明度）またはグラデーション（2色・角度） |
| 枠線 | 色・太さ・線種（実線/破線など） |
| 影 | 色・透明度・オフセット・ぼかし量 |
| フォント | フォント名・サイズ・太字・斜体・色 |

#### 操作
| 操作 | 説明 |
|------|------|
| スタイル保存 | 図形1個を選択 → 名前入力 → 保存 |
| スタイル適用 | 一覧でスタイル選択 → 「選択図形に適用（N個）」 |
| ダブルクリック | 選択図形に即座に適用 |
| お気に入り切替 | ★マーク付け・解除。お気に入りは一覧上部に表示 |
| 削除 | 選択したスタイルを削除（確認ダイアログあり） |
| エクスポート | ライブラリ全体をJSONファイルに書き出し |
| インポート | JSONファイルから読み込み（同名スタイルは上書き/スキップ選択） |

#### 永続化
- 保存先: `%APPDATA%\MagosaAddIn\StyleLibrary.json`
- 最大保存数: 100件
- アドイン再起動後も保存状態を維持

### ✂️ 図形分割機能

#### 単一図形分割
- 1つの四角形を指定した行×列のグリッドに分割
- 元図形のスタイル（塗りつぶし・枠線・影）を保持
- 水平・垂直マージンを個別設定可能

#### 複数図形グリッド分割
- 複数図形の範囲内でグリッド分割を実行
- 元図形削除の選択可能
- 四角形以外の図形に対する警告表示

### 🎯 選択補助機能

#### 選択条件
| 条件 | 説明 |
|------|------|
| 塗りのカラーコード | RGB値が同じ図形を選択 |
| 枠線のスタイル | 色・太さ・破線パターンが同じ図形を選択 |
| 塗りと枠線のスタイル | 塗りつぶしと枠線の両方が同じ図形を選択 |
| シェイプの種類 | 四角形・円・三角形など同じ種類の図形を選択 |

### 🔧 図形ハンドル調整機能

#### 対応図形タイプ
**調整ハンドル対応図形**:
- 四角形系: 角丸四角形、切り取り角、各種角処理
- 矢印系: 各種矢印、山形、矢じり調整
- 星形系: 4pt～32pt星、内側頂点位置調整
- 吹き出し: 四角形・円形・雲形の吹き出し線位置調整
- その他: 平行四辺形、台形、リボン、スクロールなど80種類以上

**角度ハンドル対応図形**:
- 円弧、弦、扇形（部分円）
- ブロック円弧（開始角度・終了角度・内径比率）
- ドーナツ（内径比率）
- 三日月（角度調整）

### 🎨 テーマカラー生成機能

#### 17種類の配色パターン

**色相ベース配色（9種類）**:
1. **Dyad（ダイアード）**: 補色関係（180°）
2. **Triad（トライアド）**: 3等分（120°）
3. **Tetrad（テトラード）**: 4等分（90°）
4. **Pentad（ペンタード）**: 5等分（72°）
5. **Hexad（ヘクサード）**: 6等分（60°）
6. **Analogy（アナロジー）**: 類似色（30°）
7. **Intermediate**: 中間色（90°）
8. **Opponent**: 対立色（135°）
9. **SplitComplementary**: 補色の分割（180°±30°）

**トーンベース配色（7種類）**:
10. **ToneOnTone**: 同色相・明度差大
11. **ToneInTone**: トーン統一・色相変化
12. **Camaieu**: 色相・トーン近似
13. **FauxCamaieu**: カマイユより色相変化大
14. **DominantColor**: 同色相統一
15. **Identity**: 1色相・明度彩度変化
16. **Gradation**: 色相・明度・彩度を段階的変化

**コントラスト配色（3種類）**:
17. **HueContrast**: 色相コントラスト
18. **LightnessContrast**: 明度コントラスト
19. **SaturationContrast**: 彩度コントラスト

## 🚀 使用方法

### 基本操作
1. PowerPointで図形を選択
2. "Magosa Tools"タブから機能を選択
3. 必要に応じてダイアログで設定を入力
4. 実行

### テキスト一括編集の使い方
1. 編集したい図形を複数選択
2. 「テキスト一括編集」ボタンをクリック
3. **テキスト編集タブ**: グリッドのテキスト列を直接クリックして編集 → 「グリッドの内容を適用」
4. **テキスト配布**: 下部テキストボックスに改行区切りでテキストを入力 → 「配布実行」
5. **検索・置換タブ**: 検索文字列・置換文字列を入力して「すべて置換」
6. **フォント書式タブ**: 変更したい項目にチェックを入れて設定 → 「フォント書式を適用」

### スタイルライブラリの使い方
1. **スタイル保存**: 保存したいスタイルの図形を1個選択 → 「スタイルライブラリ」ボタン → 「現在の図形から保存」 → 名前を入力
2. **スタイル適用**: 適用先の図形を選択 → 「スタイルライブラリ」ボタン → 一覧でスタイルを選択 → 「選択図形に適用」（またはダブルクリック）
3. **チーム共有**: 「エクスポート」でJSONファイルを保存 → チームメンバーに配布 → 「インポート」で読み込み

### 図形選択要件
- **整列機能**: 2つ以上の図形が必要
- **分割機能**: 1つ以上の四角形が必要
- **選択補助**: 1つの図形を基準として選択
- **ハンドル調整**: 1つ以上の対応図形が必要
- **レイヤー調整**: 2つ以上の図形が必要
- **ナンバリング**: 1つ以上の図形が必要
- **図形置き換え**: 記憶は1つ以上、テンプレートは1つ
- **テキスト一括編集**: 1つ以上の図形が必要
- **スタイルライブラリ（保存）**: 1つの図形が必要（適用は0個以上）
