# MagosaAddIn - PowerPoint図形操作補助アドイン

PowerPointでの図形操作を効率化するVSTOアドインです。図形の整列・分割・配置・選択補助・ハンドル調整・レイヤー管理・ナンバリング・置き換え・サイズ調整・配列複製・テーマカラー生成・テキスト一括編集・スタイルライブラリ・画像倍率同期・画像色編集・選択順序変更機能を提供します。

## 機能一覧

リボンタブ **"Magosa Tools"** のグループ構成に沿って説明します。

---

### 選択補助

#### 同一書式選択

基準図形と同じ書式を持つ図形を自動選択します。複数の選択状態をスタックで保存・復元できます。

- 塗りつぶし色・枠線スタイル・シェイプ種類での条件選択
- 選択オブジェクトスタック（最大20個の選択状態を保存・復元）
- 選択数のリアルタイム表示

**使い方**: 基準図形を1つ選択 → "選択補助" > 同一書式選択 → 条件を選択して実行

#### 選択順序変更

`ShapeRange` のZ順に依存しない論理的な処理順序を設定します。後続のレイヤー操作やスタック保存に反映できます。

- ListViewで図形一覧を確認し↑↓ボタンで並び替え
- 後続アクション：選択状態を維持 / レイヤー順に適用 / スタックに保存

**使い方**: 2つ以上の図形を選択 → "選択補助" > 選択順序変更 → 並び替えて後続アクションを選択 → OK

---

### 図形操作

#### グリッド分割

四角形を指定行×列のグリッドに分割します。元図形のスタイル（塗り・線・影）を継承します。

- 単一図形の分割（行数・列数・マージン指定）
- 複数図形の範囲内でのグリッド分割
- 元図形の書式を新しい図形に自動継承

**使い方**: 分割する四角形を選択 → "図形操作" > グリッド分割 → 行数・列数・マージンを設定 → 実行

#### レイヤー調整

図形の選択順や位置基準（左→右、上→下）で重なり順を整理します。

- 選択した順番に前面・背面へ配置
- 位置基準（左から右、上から下）で前面配置

**使い方**: 2つ以上の図形を選択 → "図形操作" > レイヤー調整 → 配置方法を選択して実行

#### 自動ナンバリング

選択図形に連番を自動挿入します。算用数字・丸数字・アルファベット・ローマ数字に対応しています。

- 開始番号・増分値・フォントサイズを指定
- 既存テキストの先頭に番号を追加

**使い方**: 番号を付ける図形を選択 → "図形操作" > 自動ナンバリング → フォーマット・開始番号を設定して実行

#### カラー生成

基準色から17種類の配色パターンでカラーパレットを生成し、スライド上に配置または図形に適用します。

- 色相ベース・トーンベース・コントラストなど17種類の配色パターン
- 明暗バリエーションを1〜10段階で展開
- 選択図形への順番またはランダム適用

**使い方**: "図形操作" > カラー生成 → 基準色・配色パターン・色数を設定 → パレット生成または図形へ適用

#### テキスト一括編集

選択図形のテキストをグリッドで一覧編集・配布・検索置換・フォント書式変更・レイアウト調整できます。

- テキスト一覧のグリッド編集・配布（改行区切りで各図形に1行ずつ）
- 検索・置換（大文字小文字区別オプション付き）
- フォント書式（フォント名・サイズ・太字・色）とレイアウト（行間・余白）の一括変更（チェック項目のみ適用）

**使い方**: 編集する図形を選択 → "図形操作" > テキスト一括編集 → タブを切り替えて操作 → 適用ボタンを押して実行

#### スタイルライブラリ

図形のスタイル（塗り・線・影・フォント）を名前付きで保存し、ワンクリックで他の図形に適用できます。

- GDI+によるリアルタイムプレビュー
- お気に入り管理・名前検索
- JSONでのインポート/エクスポート（チーム間共有対応）
- 保存先: `%APPDATA%\MagosaAddIn\StyleLibrary.json`（最大100件）

**使い方**: スタイルを保存する図形を1つ選択 → "図形操作" > スタイルライブラリ → 「現在の図形から保存」で登録 → 適用先図形を選択してダブルクリックで適用

---

### 図形置き換え

複数図形をテンプレート図形で一括置換します。サイズ・スタイル・テキストの継承オプションを選択できます。

- 元サイズ維持またはテンプレートサイズへの統一
- スタイル（塗り・線・影）とテキストの継承オプション
- 中心位置を保持して置換

**使い方**: 置き換え元の図形を選択 → "図形置き換え" > 選択完了 → テンプレート図形を選択 → 置き換え実行

---

### ハンドル調整

角丸・矢印・星形などの調整ハンドルをmm単位・度数単位で一括設定します。80種類以上の図形に対応しています。

- 調整ハンドル設定（角丸・矢印・星形など）
- 角度ハンドル設定（円弧・扇形・ブロック円弧の開始/終了角度）
- 調整ハンドルのリセット

**使い方**: 対応図形を1つ以上選択 → "ハンドル調整" > 調整ハンドル / 角度ハンドル → 値を入力して実行

---

### サイズ調整

基準サイズ適用・比率保持リサイズ・パーセント拡大縮小・固定サイズ指定など多彩なサイズ操作を提供します。

- 基準図形に合わせたサイズ統一（最大/最小/指定）
- アスペクト比を保持したリサイズ
- mm/cm/pt単位での固定サイズ指定、10〜500%のパーセント拡大縮小

**使い方**: サイズ変更する図形を選択 → "サイズ調整" から機能を選択 → ダイアログで設定して実行

---

### 配列複製

図形を円形・グリッド・線形・パス・回転コピーで複製配列します。

- 円形配列（等分配置または角度指定）
- グリッド配列（行×列）
- 線形配列（方向・個数・間隔指定）、パス配列、回転コピー

**使い方**: 複製する図形を選択 → "配列複製" > 円形配列 / グリッド配列 → 方式・個数・間隔・角度を設定して実行

---

### 画像操作

#### 画像倍率同期

2枚の画像で実寸法が分かる区間を指定することで、同じ実寸スケールにリサイズします。スキャン解像度の違いも自動補正します。

- プレビュー上のクリックまたは座標直接入力で測定区間を指定
- 測定方向（自由・水平・垂直）と単位（mm/cm）を選択
- 25〜800%のプレビューズーム、中心/左上位置保持オプション

**使い方**: スライドに画像を2枚以上配置 → "画像操作" > 画像倍率同期 → 画像①②を選択して測定区間と実寸法を入力 → 実行

#### 色調編集

Magick.NETを使用して選択画像の明るさ・コントラスト・色相・彩度・色調変換・カラーライズをリアルタイムプレビューしながら編集します。

- 明るさ・コントラスト・色相・彩度のスライダー調整
- 色調変換（グレースケール・セピア・白黒）とカラーライズ（Photoshop「色彩の統一」相当）
- 300msデバウンスによるリアルタイムプレビュー（変更前/後の比較表示）

**使い方**: 画像図形を1つ選択 → "画像操作" > 色調編集 → スライダーで調整してプレビューを確認 → OK で適用

---

### 整列

基準整列・隣接整列・拡張整列の3グループで、多彩な整列操作を提供します。

| グループ | 機能 |
|----------|------|
| 基準整列 | 左端・右端・上端・下端・水平中央・垂直中央揃え |
| 隣接整列 | 各方向の隣接配置（基準図形の端に隣接） |
| 拡張整列 | 等間隔・マージン・グリッド・円形配置 |

**使い方**: 2つ以上の図形を選択 → 各整列グループのボタンをクリック（ダイアログが開く場合は設定して実行）

## 制限事項

| 項目 | 制限値 |
|------|--------|
| 整列・レイヤー調整（最小図形数） | 2個以上 |
| 分割対象 | 四角形のみ、最大50×50 |
| 選択オブジェクトスタック | 最大20個 |
| スタイルライブラリ | 最大100件 |
| 画像倍率同期 | スライド内に2枚以上の画像が必要 |
| 選択順序変更（最小図形数） | 2個以上 |
| サイズ調整パーセント範囲 | 10%〜500% |
| 配列個数範囲 | 2〜100個 |
